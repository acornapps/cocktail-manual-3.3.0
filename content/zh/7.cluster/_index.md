提供集群的注册管理，以及节点和应用程序为单位的监控、卷、报警和计量服务。

![]({{< param imageRootDir >}}/assets/ZH/{{< param version >}}/7_1.png)

| **集群菜单** | **说明** |
| :--- | :--- |
| 节点 | 所有节点的 CPU、内存、网络使用量以及个别节点的监控服务 |
| 应用程序 | 所有应用程序的 CPU、内存、网络使用量以及个别应用程序的监控 服务 |
| 卷 | 存储管理以及卷现状 |
| 报警 | 集群事件信息相关的报警服务 |
| 计量 | 集群费用及各种资源费用服务 |
| 注册管理 | 集群的查看、修改、删除 |

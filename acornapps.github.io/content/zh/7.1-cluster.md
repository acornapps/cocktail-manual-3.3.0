注册、编辑、删除从供应商那里得到分配的 Kubernetes 集群信息的功能。

### a\) 点击集群→"排列"按钮，选择需要的排列标准。 ![]({{ site.baseurl }}/assets/ZH/{{ site.version }}/7.1_1.png)

| **集群设置** | **说明** |
| :--- | :--- |
| 集群 | 集群的种类 |
| 节点 | 集群的节点数字 |
| CPU | 集群所有的 CPU 容量、请求量、使用量 |
| 内存 | 集群所有的内存容量、请求量、使用量 |
| 应用程序 | 集群的应用程序表(命名空间)数 |
| 工作负载 | 正在部署在集群的工作负载数 |

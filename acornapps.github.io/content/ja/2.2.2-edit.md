クラスターの基本情報、プロバイダー、モニタリング、クラスターのタイプを変更する機能である。 プロバイダーのアカウント、タイプ、リージョンは変更できない。

### a\) クラスター → クラスター選択 → 登録管理 → 変更ボタン → 修正完了後、保存ボタンをクリック。

---
cluster list는 Cocktail DashBoard 또는 Cocktail Online에 등록된 Cube Cluster를 정보를 표시한다. 

---

### Prerequisite
Cube Cluster 설치 상태.`create` command 

Cocktail DashBoard 또는 Cocktail Online에 로그인된 상태. `login` command 

### Usage
* `cube cluster list`
